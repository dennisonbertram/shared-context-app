# Context Archive

> Aggregated prompt/context bundles that were previously sitting in the repository root. These are now centralized so future contributors can find historical exports without cluttering the workspace.

| File | Description |
| --- | --- |
| `delivery-summary.txt` | Snapshot of the original delivery summary shared with reviewers. |
| `docs-complete-context.txt` | Full documentation context dump (initial pass). |
| `docs-complete-context-2025-01-16.txt` | Timestamped update of the complete docs context. |
| `gpt5-adr-feedback.txt` | Consolidated feedback from GPT-5 on ADRs. |
| `gpt5-mcp-server-review.txt` | GPT-5 review of the MCP server implementation. |
| `mcp-server-review-context.txt` | Context package that accompanied the MCP review. |
| `project-context-arch.txt` | Architecture-focused project context export. |
| `project-context-decisions.txt` | ADR-focused project context export. |
| `project-context-guides.txt` | Guides-focused project context export. |
| `project-context-plans.txt` | Plans-focused project context export. |
| `project-context-reference.txt` | Reference-focused project context export. |

For any new large prompt/context bundles, drop them in this directory (or a subfolder) and link them here so the root stays clean.
